# Contributing

Thanks for your interest in `@yca-software/yca-react-core`.

## Development Setup

1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Run lint checks:
   ```bash
   pnpm lint
   ```
3. Run tests:
   ```bash
   pnpm test:run
   ```
4. Run Storybook locally:
   ```bash
   pnpm storybook
   ```

## Pull Request Guidelines

- Keep changes focused and small.
- Add or update tests for behavior changes.
- Update `README.md` for new components, props, or workflow changes.
- Ensure CI is green before requesting review.

## Commit and Release Notes

- Write clear commit messages describing the intent.
- For user-facing changes, include a short note in the PR description about migration or usage impact.

## Security Reports

Security reports are welcome. Please use `SECURITY.md`.
